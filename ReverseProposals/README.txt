#REVERSE PROPOSALS
A SMAPI mod that adds a couple of relationship actions and tokens for use with Marry Me: Reverse Wedding Proposals

##:: REQUIREMENTS ::
+ SMAPI

##:: MOD AUTHORS ::
This mod now operates on a white list basis. You need to explicitly enable your NPC for the default proposals to play.

Please see the github page for detailed steps on how to do this, as well as guidelines for creating your own custom events.

https://github.com/Kantrip-Mods/SDV/tree/main/ReverseProposals
